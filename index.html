SKU B91627 — OCI Object Storage Requests
Ogni unità contrattualizzata = 10.000 request/mese. Il cliente ha contrattualizzato 1.000 unità → 10.000.000 req/mese. Inserisci la banda e i parametri Veeam: il tool stima i dati trasferiti al giorno/settimana/mese e le request generate, espresse in unità B91627.

Storage Optimization — Blocchi e Dedup
La dedup Local (source-side) riduce i blocchi trasmessi sulla WAN: incide direttamente sulle request generate. La dedup WAN target (OCI-side) riduce solo lo spazio occupato su OCI: non riduce le request. Le request dipendono dai blocchi fisicamente trasmessi dopo il local dedup.
KB
Blocchi più grandi = meno request/TB. 256KB (LAN) · 512KB (WAN) · 1024KB (WAN ottimizzato).
x
Riduce i blocchi inviati sulla WAN. 1x = nessuna riduzione. Tipico: 1.5x–3x.
x
Riduce solo lo spazio OCI occupato, non le request. Usato per la stima storage finale.
